CSS Syntax

Syntax Use Example
Selector ตัวที่กำหนดว่าโค้ด css ต่อจากนี้จะถูกใช้งานในส่วนไหนของไฟล์htmlบ้าง .div #id
Property กำหนดpropertyของselectorนั้น เช่นเลือกสีเลือกไซส์เลือกรูปแบบ color: font-size:
Value ค่าของpropertyที่ถูกกำหนดโดยvalueนั้นจะมีแบบที่ต่างกันbased on property color:Black font-size:12px

การใช้ CSS ในแบบต่างๆ

แบบ Explanation
เขียนในHtml file เขียนcss style ในfile htmlเลยโดยไม่มีความจำเป็นต้องมีreferenceไปที่ไฟล์css โดยปกติแล้วจะเขียน <style>css</style> ใน headของhtml
เขียนในCss file สร้างfile.cssขึ้นมาและเขียนโค้ดลงไปในนั้น โดยต้องทำการเชื่อมกับhtml fileโดยใช้ src hrefไปที่ไฟล์.cssที่มี

Selectors

selectors use
.class เลือกเฉพาะclassนั้นๆ โดยแต่ละคลาสสามารถมีได้หลายส่วน
#id เลือกเฉพาะidนั้นๆ แต่idนั้นจะได้เฉพาะส่วนๆเดียว
* เลือกelementทั้งหมด
element เลือกเฉพาะelement เช่น div หรือ h1
element, element เลือกหลายๆelement เช่น div, h1, h2

Combinators

Combinator use
element1 element2 เลือกelement1ทั้งหมกที่อยู่ในelement2
element1 > element2 เลือกelement1ทั้งหมดที่เป็น child ของ element2
element1 + element2 เลือกelement1ที่ต่อจากelement2
element1 ~ element2 เลือกประเภทelement1ทั้งหมกที่ตามหลังelement2

Property and Value

Property vs Value

Property นั้นเป็นตัวบอกค่าที่จะเซ็ทให้กับelement เช่นสีหรือไซส์ ส่วนตัวvalueนั้นเป็นตัวกำหนดว่าค่าต่างๆจะมีค่าเท่าไหร่่เช่นให้สีเป็นสีดำ หรือให้ไซส์เท่ากับ20pt
